# Project Statement
"Student Result Management System"

## Problem Statement
Colleges need a simple way to store student marks and calculate results.
Many times marks are written on paper or in different files,making it difficult to manage.
This project provides a simple system where we can add students,Update marks,calculate results and store everything in one file.

## Scope of the Project
-This system works through a simple menu on the screen .
-It stores all student data in a CSV files so the information is saved permanently.
-The user can add,search,update,view and delete student records.
-The system automatically calculates total marks,percentage and grade.

##Target users
-Teacher who want to store marks easily.
-Students who want to understand how Python file handling works.
-Anyone learning basic Python projects.

## High-Level Features
-Add new student with marks in 5 subjects.
-View all stored students.
-Search students by ID or name.
-Update marks of an existing student.
-Delete a student record.
-Automatic calculation of:
    -Total marks
    -Percentage
    -Grade(A+,A,B,C,D,F)
-Data stored safely in CSV format.
