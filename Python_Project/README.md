Students Result Management System

##Overview
A console_based Python application to manage student records(add,view,search,update,delete).
Data is stored in a CSV file(Comma Separated Values )
This system calculates total marks,percentage,and grade automatically.

##Features
-Add student with 5 subject marks
-View all students
-Search by student ID or name
-Update student marks
-Delete student records
-Data persisted in students.csv
-Unit tests included

##Requirements
-Python 3.8+
