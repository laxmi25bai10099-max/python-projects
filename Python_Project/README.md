Students Result Management System

##Overview
This project helps you easily manage student results from the terminal 
You can add new students, view all records, update marks, search for specific students, and delete records when needed

##Features
-Add a new student with marks of 5 subject 
-View all student records
-Search student by ID or name
-Update  existing student marks
-Delete student records
-Automatic data storage in students.csv
-Unit tests included
-Automatic total, percentage, and grade calculation
-Topper Identification:
    The system can automatically find the student with student with the highest total marks.
    -Name
    -Student ID
    -Total Marks
    -Percentage
    -Grade

##Requirements
-Python 3.8 or higher

##Run
1.Open terminal in project folder
2.Run:
--Type: python main.py
The main menu will appear with all available options. choose accordingly.
